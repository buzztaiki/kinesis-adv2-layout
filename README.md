# My Kinesis Advantage2 Keyboard layout

## How to apply:
1. enter v-drive mode (type `program + F1`)
2. run `make`
3. exit v-drive mode (type `program + F1`)

## Cheat Sheat

```

*tap and hold: tap to esc, hold > 250ms to lshift

* macro
```
